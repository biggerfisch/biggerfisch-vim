biggerfisch-vim
===============

This is a collection of vim plugins that I enjoy using.

Install
-------
'''
git clone git@github.com:/biggerfisch/biggerfisch-vim
cd biggerfisch-vim
git submodule init
git submodule update
./install_pathogen.sh
./link_rc.sh
'''
